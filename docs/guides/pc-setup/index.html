01

BTOパソコンの開封と準備

パソコン工房の「LEVEL-R8B8」シリーズは高性能な分、輸送時の衝撃を防ぐためにケース内部に緩衝材が詰められていることがあります。電源を入れる前に必ず確認してください。

致命的なエラーを防ぐために

ケース内部の緩衝材を取り除かずに電源を入れると、ファンが回転できず発熱によりパーツが破損する絶大なリスクがあります。左側面のガラスパネルのネジを外し、中身を確認してください。

  • 箱から取り出し、安定した平らな場所に置きます。（床の直置きはホコリを吸うため避けるのがベストです）
  • ケースを開け、内部のクッション材や固定テープをすべて取り除きます。
  • 付属の電源ケーブルをコンセントとPC背面の電源ユニット（最下部）に繋ぎ、スイッチを「I（オン）」に入れます。
02

Pixio PS2S モニターアームの設置

デュアルモニターアーム「Pixio PS2S」はガススプリング式のため、モニターを浮かせたあとに張力（テンション）の調整が必要です。

  • クランプの固定: デスクの天板にアームの土台を挟み込み、下から六角レンチでしっかりと締め付けます。（天板の厚みに注意してください）
  • VESAマウントの取り付け: モニターの背面に付属のVESAプレート（四角い金具）を4つのネジで固定します。
  • アームへの装着: モニターをアームの先端にスライドさせてカチッと嵌め込みます。この時、モニターが重くてアームが下がってしまう場合は、アーム関節部にある「＋/－」のネジを六角レンチで「＋（プラス）」方向に回してガス圧を強めます。
  • ケーブルマネジメント: アームのプラスチックカバーを外し、電源ケーブルとDisplayPortケーブルを中に這わせることでデスク上がスッキリします。
03

ケーブルの接続（超重要）

初心者が最も陥りやすい罠が「モニターの接続先」です。映像ケーブルは必ずグラフィックボード（下部の横向きの端子）に接続してください。

マザーボードの端子（縦向き） ❌ 挿さない
グラフィックボードの端子（横向き）
  • モニター付属のDisplayPortケーブル（またはHDMI）をグラボとモニターに繋ぎます。（デュアルモニターの場合は両方ともグラボへ）
  • キーボードとマウスを、マザーボード背面にあるUSB端子に繋ぎます。
04

ASUS RT-BE72 (Wi-Fi 7)のネットワーク構築

最新規格のWi-Fi 7ルーターを使用して、安定したゲーミングネットワークを構築します。

  • 配線: モデム（壁からの回線機器）からのLANケーブルを、RT-BE72背面の「WAN」ポート（青色など色分けされている端子）に接続します。
  • 電源投入: ルーターの電源プラグをコンセントに差し込み、電源ボタンを押します。LEDランプが白または青に点灯するまで数分待ちます。
  • 有線接続: ゲーミングPCは安定性のため、ルーターの「LAN」ポートのいずれかからPC背面のLANポートへ有線（LANケーブル）で繋ぐことを強く推奨します。
  • アプリ設定（Wi-Fi用）: スマホに「ASUS Router」アプリをインストールし、ルーター底面に記載された初期Wi-Fi(SSID)に繋ぐと、ワンタップで初期設定（新しいパスワードの設定など）が完了します。
05

Steamの導入とゲーム用ストレージ設定

最後に、PCゲームのプラットフォームである「Steam」を導入します。BTOパソコンではOS（Cドライブ）とデータ用（Dドライブ等）が分かれている場合があるため、容量の大きいドライブをゲームの保存先に設定します。

  • インストール: Steam公式サイトからインストーラーをダウンロードし、インストールします。
  • ログイン: アカウントを作成（無料）するか、既存のアカウントでログインします。
  • ストレージの追加設定: 左上の「Steam」メニュー ＞「設定」＞「ストレージ」を開きます。
  • 画面上部のドライブ名の横にある「＋」アイコンをクリックし、Dドライブ（大量の空き容量があるドライブ）を選択して新しいライブラリフォルダを作成します。
  • 追加したDドライブの横の「…」アイコンをクリックし、「デフォルトにする」を選択します。これで今後ダウンロードするゲームは自動的に容量の大きいドライブに保存されます。
