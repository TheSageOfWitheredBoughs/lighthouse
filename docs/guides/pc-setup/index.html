01

LEVEL-R8B8 (RGB Build) の開封と設置

Ryzen 7 9800X3D + Radeon RX 9070 XT 構成

パソコン工房の「LEVEL-R8B8-LCR98D-TGX」は、大型グラフィックスカードと水冷クーラーを搭載した高性能モデルです。ガラスパネルモデル特有の注意点や、輸送中のパーツ保護材の撤去手順を確実に実行してください。

電源を入れる前の必須作業：ケース内部の緩衝材撤去

大型グラボ（RX 9070 XT）を輸送中の脱落から守るため、ケース内部に発泡ウレタンやエアパッキンなどの緩衝材が詰められています。これを取り除かずに電源を入れると、ファンが回転できず発煙や故障の原因となります。

Hover to open glass panel

  1. ガラスパネルの取り外し: 本体左側面（マザーボードが見える側）のネジを手で回して外し、ガラスパネルを慎重に取り外します。破損に注意してください。
  2. 緩衝材の完全除去: グラフィックボードの下やファン周辺に詰まっている保護材やテープをすべて取り除きます。
  3. 電源ケーブルの接続: 付属の太い電源ケーブルをPC背面の最下部にある電源ユニットに挿し込み、電源ユニットのスイッチを「I（オン）」側に倒します。

映像ケーブルの接続箇所（最も多いミス）

モニターアームに設置したモニターとの接続です。マザーボード上の端子（縦向き）は使用せず、必ず増設されたグラフィックボードの端子（横向き・下部）を使用してください。

❌ マザーボード（縦）には挿さない

※ここに挿すと画面が映らない、またはグラボの性能が発揮されません

✅ グラフィックボード（横）に挿す

DisplayPort または HDMI ケーブルを接続

02

Pixio PS2S デュアルモニターアーム設置

ガススプリング式の組み立て・調整

Pixio PS2Sはガススプリングを内蔵した多関節のデュアルモニターアームです。モニターを取り付けた後の「張力（テンション）調整」が最も重要なポイントになります。

  1. デスクへの土台（ベース）固定:
    クランプ式の場合、デスクの天板をC字型の金具で挟み込み、下部のノブを回してしっかりと固定します。天板の厚みが不足する場合は、間に補強板（別売）を挟むことを推奨します。
  2. VESAマウントの取り付け:
    モニターのスタンドを外し、背面のネジ穴（100x100mmまたは75x75mm）に付属のVESAプレートをネジで固定します。対角線上に均等に締めるのがコツです。
  3. アームへの装着:
    モニターに取り付けたVESAプレートを、アーム先端のブラケットに上からスライドさせてカチッと嵌め込みます。その後、ロック用のネジを締めて脱落を防ぎます。
  4. ガススプリングのテンション調整:
    モニターを浮かせた際、モニターの重さでアームが下がってしまう場合は、アームの関節部にある調整ネジを付属の六角レンチで「＋（プラス）」方向に回します。逆にアームが跳ね上がってしまう場合は「－（マイナス）」方向に回します。水平で止まる位置が適正テンションです。
ケーブルマネジメントについて

アームの下部にはプラスチック製のケーブルカバーが付いています。カバーをスライドさせて外し、電源ケーブルとDisplayPortケーブルをアームに沿わせることで、デスク上が驚くほどすっきりします。ケーブルにはアームが動く分の「ゆとり」を持たせることを忘れないでください。

03

ASUS RT-BE72 (Wi-Fi 7) ネットワーク構築

超低遅延・高速ゲーミング回線の設定

最新規格「Wi-Fi 7」に対応したハイエンドルーター ASUS RT-BE7200 をセットアップし、BTOパソコンへ最高の通信環境を提供します。

モデムの再起動手順を絶対に省かないでください

既存のルーターから ASUS RT-BE72 に交換する場合、壁から繋がっている大元の機器（モデム／ONU）が以前のルーターの情報を記憶しているため、そのまま繋いでもインターネットに繋がりません。モデムの電源コンセントを必ず一度抜き、1分待ってから再度挿してください。

  1. モデムとルーターの有線接続:
    モデム（再起動済み）のLANポートと、ASUS RT-BE72 背面の「WAN」ポート（青色、または地球マークのある一番端のポート）を付属のLANケーブルで接続します。
  2. ルーターの電源投入:
    RT-BE72 をコンセントに繋ぎ、背面の電源スイッチを押します。前面のLEDランプが点滅から白の単色点灯（またはインターネット接続完了を示す色）に安定するまで数分待ちます。
  3. PCとの有線LAN接続（推奨）:
    ゲーミングにおいて遅延（Ping）は勝敗を分けます。RT-BE72 の「LAN」ポート（黄色など）から、BTOパソコン背面のLANポートへ有線ケーブルで接続してください。（無線Wi-Fi 7機能はスマホやタブレット等のために設定します）
  4. 「ASUS Router」アプリによる初期設定:
    スマートフォンに専用アプリ「ASUS Router」をインストールします。ルーター底部のシールに記載された初期Wi-Fi (SSID) にスマホを繋ぐと、アプリが自動でルーターを検出し、新しい管理者パスワードや任意のWi-Fiネットワーク名を設定する画面（Quick Internet Setup）に進みます。
04

Steam環境とストレージ設定

数百GBのゲーム容量を賢く管理する

BTOパソコンでは、OSがインストールされている「ローカルディスク(C:)」の他に、データ保存用の広大なドライブ「ローカルディスク(D:)」などが搭載されている場合があります。ゲームデータでCドライブをパンクさせないための必須設定を解説します。

  1. Steamクライアントのインストール:
    Steam公式サイト（store.steampowered.com）の右上「Steamをインストール」からインストーラーをダウンロードし、実行します。アカウントを新規作成してログインします。
  2. 設定画面（ストレージマネージャー）を開く:
    Steamアプリ左上の「Steam」メニューから「設定（Settings）」を開き、左メニューの「ストレージ」項目をクリックします。
  3. 新しいライブラリドライブの追加:
    画面上部にあるドライブ名（例: Local Drive (C:)）の横の「＋（ドライブを追加）」ボタンを押します。プルダウンから容量の空いているドライブ（例: D:\）を選択し、「追加」を押します。
  4. デフォルトのインストール先にする:
    追加されたDドライブの右端にある「…（設定）」アイコンをクリックし、「デフォルトにする」を選択します。★マークがDドライブに移動すれば完了です。
